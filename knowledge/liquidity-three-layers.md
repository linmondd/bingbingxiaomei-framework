# 流动性三层模型

**一句话**：宏观央行 → 中观机构 → 微观游资散户，钱的方向决定一切。

## 核心观点

流动性不是单一变量，而是三层结构：

- **宏观层（央行/汇率）**：央行货币政策、汇率变化、全球资本流动——决定"钱的总量"
- **中观层（机构配置）**：公募基金、保险、北上资金、ETF 资金流——决定"钱去哪"
- **微观层（游资/散户）**：短线资金、两融余额、散户情绪——决定"钱的节奏"

三层流动性必须同向时，市场趋势才是健康的。如果宏观宽松但中观机构在减仓，或者中观在买但微观散户在恐慌，都会出现"有钱但没涨"或"涨了但没跟"的错位。

## 在体系中的位置

流动性三层模型是冰冰小美**市场结构层**的核心框架。它把抽象的"流动性"拆成可观察的三层信号，每一层对应不同的数据源和观察窗口。

## 关键论据

- 第三方知识库将流动性列为核心驱动层，三层结构是其展开方式
- "钱的流向决定一切"是体系中的核心原则
- 2026-03-20 宏观风险清单可作为宏观层的具体观察指标
- 与亏钱效应互补：亏钱效应看风险扩散，流动性看资金方向

## 推理链

央行政策 → 汇率与利率变化 → 机构配置调整 → 板块资金流 → 散户跟风 → 赚钱/亏钱效应 → 反馈到下一轮

## 反证与风险

- **反证**：不同市场阶段三层权重不同，不能固定套用
- **风险**：宏观流动性宽松不必然带来微观赚钱效应——中间隔着"钱去哪了"的问题
- **内部张力**：三层方向持续背离时，需要拆开解释而非强行合并
