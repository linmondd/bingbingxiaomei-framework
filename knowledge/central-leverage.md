# 中央加杠杆

**一句话**：中央加杠杆是财政、央行、化债和新产业投资的关键承接机制。

## 核心观点

中央加杠杆是冰冰小美体系中反复出现但尚未完全恢复的核心概念。它的基本逻辑是：当地方政府、企业和居民的杠杆空间有限时，中央政府的资产负债表成为化解存量债务、推动新产业投资的最后承接方。

这不仅是财政问题，更是央行与财政分工的重新定义——谁来承担转型成本、谁来为新产业提供初始资本、谁来做"最后借款人"。

## 在体系中的位置

中央加杠杆位于冰冰小美体系的**宏观世界观层**，与国运框架、AI-美元战略并列。它是连接财政政策、货币政策、产业政策和资本市场的中枢概念——钱从中央来，经过金融体系，流向新产业。

## 关键论据

- 体系整理中反复出现，是解释化债与转型的核心线索
- 与"金融市场三大功能"中的"融资""纾困"功能直接关联
- 涉及央行公开透明、财政扩张、科技投资和股市融资功能的联动

## 推理链

中央财政扩张 → 承接地方/企业债务 → 释放银行资产负债表 → 为新兴产业提供融资空间 → 资本市场重定价 → 新旧经济轮动
