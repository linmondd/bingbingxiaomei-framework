# 当支柱产业逻辑失效：从汽车股暴跌看宏观风险与产业周期切换

**一句话**：宏观风险＞中观风险，支柱产业逻辑需随产业周期与政策变化动态调整。

## 核心观点

1. 2023年基于“汽车是支柱产业”的逻辑买入汽车股（如比亚迪），但随着2025-2026年汽车销量增速从预期15%骤降至7%，以及国补减半、安全事故等冲击，该逻辑已失效。
2. 市场对支柱产业的定价从高增长（40%增速）转向低增长（7%），导致股价大幅回调；即使降低预期（15%增速）买入，仍因实际增速远低于预期而亏损。
3. 宏观风险（如石油危机、补贴政策变化）对行业的影响大于中观风险（如行业销量下滑），且宏观信息公布时往往已晚，投资者需在宏观风险节点控制仓位。
4. 产业周期切换：2023年支柱产业（汽车）是主线，但2026年新兴产业（如电池新国标、出海）成为新增长渠道，市场更愿意相信新兴产业，配置需从支柱产业转向新兴产业。

## 关键论据

- 比亚迪股价从400元（按40%增速计价）跌至310元（按15%增速计价），再跌至90元（按7%增速），实际增速远低于预期。
- 2025年4月汽车安全事故引发销量下滑，叠加国补减半，加剧了行业悲观情绪。
- 2026年7月新电池国标落地，可能导致比亚迪旧电池资产折旧亏损，进一步压制股价。
- 赛力斯等销冠车型的暴跌传导至整个汽车板块，市场极端悲观与宏观数据下滑叠加。

## 推理链

1. **政策驱动**：2023年国补+支柱产业政策 → 汽车销量高增长 → 市场给予高估值（40%增速）。
2. **中观冲击**：2025年安全事故+国补减半 → 销量增速从15%骤降至7% → 估值逻辑崩塌。
3. **宏观风险**：石油危机、出海冲击等宏观因素 → 加剧国内内卷 → 销量下滑超预期。
4. **产业切换**：2026年新电池国标落地 → 旧资产减值风险 → 市场转向新兴产业（如电池、出海） → 支柱产业逻辑失效。

## 反证与风险

- **反证1**：如果汽车销量增速在2025-2026年维持15%以上（如出海爆发式增长），则支柱产业逻辑仍成立，比亚迪股价不会大幅下跌。
- **风险1**：该观点假设宏观风险（如石油危机、补贴政策）是主要矛盾，但若中观风险（如行业竞争、技术迭代）成为主导，则宏观风险控制策略可能失效。
- **风险2**：新兴产业（如电池、出海）的估值泡沫可能同样面临政策或技术风险，切换配置可能带来新的亏损。

## 观察指标

- **汽车月度销量增速**：是否持续低于10%（验证支柱产业逻辑失效）。
- **比亚迪新电池国标落地后的资产减值公告**：是否出现大额折旧亏损。
- **新兴产业（如电池、出海）的季度营收增速**：是否持续高于20%（验证产业切换）。
- **宏观风险指标**：石油价格、补贴政策变化、贸易摩擦等。

## 适用边界

- **什么时候适用**：当支柱产业面临政策退坡、安全事故等中观冲击，且宏观风险（如石油危机）叠加时，该框架有效。
- **什么时候不适用**：当支柱产业处于政策强支撑期（如2023年国补高峰），或宏观风险可控时，该框架可能过度悲观。
- **框架的局限性**：依赖宏观风险判断，但宏观风险难以预测；对中观风险（如行业竞争）的量化不足，可能低估内卷程度。
