# 冰冰小美框架

Copyright (c) 2026 Mon.

冰冰小美框架是一个证据型人物知识体系 Skill，用于整理、检索和分析雪球用户「冰冰小美」的公开观点。它不是把若干金句拼成角色提示词，而是把公开材料拆成可查询的观点卡，记录来源、发布时间、证据等级、历史背景、反证、内部张力和失效条件。

本项目不代表「冰冰小美」本人，不声称获得其授权，不构成投资建议、交易建议、收益承诺或任何形式的带单服务。所有观点分析都应回到原始公开来源、发布时间、市场背景和证据等级中理解。

## 能做什么

- 用证据等级检索已整理观点的原帖、日期、来源状态和限制。
- 区分作者原话、第三方整理、组织者归纳和 Skill 推断。
- 保留删帖、锁帖、评论、图片、直播或未索引内容造成的 U 级缺口。
- 对当前市场先核验现实事实，再映射历史框架。
- 输出观点的反证、风险边界和失效条件。

## 当前覆盖

当前公开版包含 20 个结构化观点节点、16 个来源登记、3 个未解来源任务、参考文档、查询脚本和测试。覆盖宏观世界观、市场结构、产业映射、交易体系、方法反思与来源地图。

这不是完整全集。评论、图片、直播、播客、转发附言、删帖、锁帖和未索引内容仍可能缺失。项目不得宣称“已经收齐全部观点”。

## 证据等级

| 等级 | 含义 |
|---|---|
| A | 已直接打开并核验一级来源全文或足够上下文 |
| B | 一级来源 URL、状态 ID、标题、时间可由搜索索引与用户侧整理材料交叉验证，但页面无法稳定完整读取；搜索索引片段最高只能进入 B 级 |
| C | 两个以上相互独立的二手来源一致复现 |
| D | 单一用户侧整理材料、播客转制或第三方知识库复现 |
| U | 没有可靠出处，或来源可能删除、锁定、仅在评论/图片/直播中出现 |

证据等级描述的是“这句话能否归到作者及其原始语境”，不是观点是否正确。

## 来源边界

本项目只保留必要的来源索引、短摘录、结构化观点卡、证据等级、反证条件和使用边界，不收录、不再分发、不替代原作者完整文章。

第三方 GitHub 仓库、公开知识站和搜索结果仅用于来源发现和交叉验证。若第三方仓库没有许可证，本项目不会复制其代码、配置或大段正文。
